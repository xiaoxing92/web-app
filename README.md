## ⚡ 简介

# 项目目录

├─src 主目录
│ ├─api 请求 Api
│ ├─assets 静态资源
│ ├─components 公用组件  
│ ├─hooks 组件 hooks
│ ├─router 路由
│ ├─utils 工具
│ └─views 页面

```bash
# 配置
1. node 版本 16+

# 进入项目目录
cd web-app

# 安装依赖
npm i

# 启动服务
npm run dev
```

## 功能

- **搜索**：按回车键进行搜索
- **搜索范围**：精确查准
- **Paw**:支持

## 存在问题
